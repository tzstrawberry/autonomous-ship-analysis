# autonomous-ship-analysis
Development of a Python-based data analysis pipeline for processing sensor data from the autonomous research vessel “Boris Kluge”, aiming to compute and analyze key operational metrics such as energy consumption (Wh/km), data density (GB/h), and environmental conditions.

Entwicklung einer Python-basierten Analysepipeline zur Verarbeitung von Sensordaten eines autonomen Forschungsschiffs („Boris Kluge“), mit dem Ziel, zentrale Betriebskennzahlen wie Energieverbrauch (Wh/km), Datendichte (GB/h) und Umweltbedingungen zu berechnen und zu analysieren.
